# Agile Query 介绍

Agile Query 是一款面向企业的自助式数据分析系统，相比传统BI 系统，Agile Query 不仅可以节约大量数据开发成本，同时还能应对日益复杂的数据分析需求。
Agile Query 内部集成了一个强大的SQL 编译器，能够根据不同的数据库表关联关系，自动构建出最优的查询SQL，同时还封装了大量领域查询函数，
使得业务人员和数据分析师能够自主地完成数据分析工作。

- **智能多表查询**：通过自动识别表之间的关联关系，智能多表查询能够生成最优的 SQL 语句，避免手动编写复杂的连接操作。
- **多表计算公式**：通过定义涉及不同表中字段的公式，用户可以实现复杂的数据运算，如聚合、统计分析和数据转换。
- **嵌套聚合公式**：允许在一个聚合函数内部再嵌入另一个聚合函数，使得数据分析师和数据库管理员能够执行多层次的数据汇总和计算，从而揭示数据的深层趋势和模式。 
- **多维度计算公式**：多维度计算公式是一种在数据分析中，跨越多个维度对数据进行计算和汇总的方法。 通过在多维数据集中定义计算成员（Calculated Member），用户可以创建涉及不同维度的公式，实现复杂的数据运算。 
- **动态分类**：基于多个聚合数据的规则，对指定维度进行分类，实现客户画像或标签，商品分类等复杂分类规则。这种方法利用多维度分析技术，结合业务需求，构建精细化的标签体系，帮助企业深入了解客户特征和商品属性，从而制定精准的营销策略和优化产品分类。
- **增长率分析**：通过内置函数，实现各种时间周期的增长率分析，如同比、环比、环比增长率等。同时，可以组合其它维度进行多维度的增长率分析，满足复杂的数据分析需求。
